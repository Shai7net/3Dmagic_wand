# מסמך פרויקט - מטה קסמים וסרטונים 3D

## מהות הפרויקט
אפליקציה המציגה בחלל תלת מימדי (3D) מטה קסמים מציאותי וזוהר באמצע המסך. סביב המטה מסתובבים קרוסלת סרטונים אקראיים אשר נשאבים באופן דינאמי משני ערוצי יוטיוב ספציפיים. 

## מה שבוצע ועודכן בפרויקט (פירוט טכני):

1. **הקמת שרת Backend מותאם (Node.js/Express):**
   - נוצר קובץ קוד `server.ts` שפועל כשרת גיבוי API. השרת מבצע קריאות רשת ישירות לערוצי היוטיוב שסופקו (`@shaitt1137` ו-`@the88creator`).
   - השרת קורא את עמודי היוטיוב ושולף באמצעות ביטויים רגולריים (Regex) את המזהים (Video IDs) של הסרטונים.
   - רשימת הוידואים נשמרת בשרת באובייקט מטמון ומנותבת בחזרה ל-Client-Side (ממשק המשתמש), כך שאם עולים סרטונים חדשים לערוץ, הם יכנסו למאגר.

2. **בניית תצוגת התלת מימד (3D Engine) בממשק - פתרון החיתוכים:**
   - **תיקון בעיית ה"הסתרה" וחיתוך במרחב (Z-index vs 3D):** בסבב הקודם, הסרטונים "שכבו עליו" כיוון שמטה הקסמים קיבל קיבוע `z-index` מעל להוויה התלת מימדית. בקוד החדש, המטה והסרטונים אוחדו לקונטיינר אב יחיד החולק את אותו מרחב וירטואלי (`preserve-3d`). כעת מערכת הרינדור (הדפדפן) מבינה שהסרטונים שמקדימה צריכים להסתיר את המטה, אך כשהם ממשיכים לאחור האנימציה - המטה מסתיר אותם בדיוק כפי שקורה בעולם האמיתי.
   - הרדיוס התלת מימדי (ציר ה-Z) של מסלול סיבוב הסרטונים הורחב משמעותית לכ-`550px`, מה שנותן מרווח נשימה נכון ותחושת ריחוף עמוקה יותר מסביב לעמוד המטה.

3. **עיצוב ומיקום מחודש של מטה הקסמים (Art Direction):**
   - כפי שביקשת, האור של מטה הקסמים מוקם בדיוק במרכז האנכי והאופקי של החלון. ה"מקל" גולש במרכז המסך מהאור עצמו ועד התחתית.
   - **שילוב מודל GLB (תלת-מימד):** המקל בעיצוב ה-CSS הומר לחלוטין למודל תלת-מימדי אמיתי (פורמט GLB) באמצעות ספריית `@react-three/fiber` ו-`@react-three/drei`. כעת, קובץ ה-GLB שעיצבת (Dmagic_wand model.glb) נשאב ישירות מהריפוזיטורי שלך ב-GitHub ומוצג בסצנת ה-React לצד האורות השונים. הוספתי למקל גם הנפשת ריחוף וסיבוב איטי כדי להשלים את הקסם.
   - **הילת האור (Glowing Orb):** שופרה דרמטית. ההילה מורכבת מ-5 שיכבות שונות של אור נושם (Pulse), מנקודה חזקה צהובה-לבנה במרכז שמתפרסת לרדיוס תאורה רחב ועמום שזורק צהוב על כל הרקע של החלון. בנוסף שוחררו חלקיקי אור נוצצים שחגים מסביב לכדור הזוהר כאפקט קסם מרחף.

4. **לוגיקת האנימציה וממשק משתמש (UX):**
   - אנימציית ה-CSS התלת-מימדית מסתובבת רצוף ב-360 מעלות במחזור של 20 שניות.
   - ב-React נכתבה פונקציה שעוקבת אחרי כל מחזור סיבוב (Animation Iteration). כל 3 סיבובים שלמים, נשלפים סרטונים באופן חסר סדר מחדש, והממשק מרענן את 5-6 הקליפים שעל המסך לסרטונים חדשים ורנדומליים מתוך המאגר.
   - נוספו כפתורי פאנל צד למטה: אפשרות לעצור או להמשיך את הסיבוב (Play/Pause), וכפתור לחילוף הסרטונים באופן ידני כעת, ללא צפייה בהמתנה של ה-3 סיבובים.
